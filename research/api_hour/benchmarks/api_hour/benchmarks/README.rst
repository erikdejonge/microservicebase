benchmarks
==========

Start manually
--------------

In this current folder, launch: `api_hour -ac benchmarks:Container`

Install
-------

#. Create a pyvenv: `~/.pythonz/pythons/CPython-3.4.2/bin/pyvenv pyvenv`
#. Enable pyvenv: `. pyvenv/bin/activate`
#. Install: `pip install -r requirements.txt`
